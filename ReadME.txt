How to use key.bat

1. first you have to change and create your own password:

you can easily change password open "key.bat" in notepad.
search '0x00'(which is the default password) wherever written.
in place on 0x00 delete and type your password and save it.
all done.

2. its time to use:

put this script in folder where you want to lock and hide your files.
example: drive d > photos.
then run "key.bat" a new folder with name: "locker" will be created automatically.
put your stuff(any files like images, music, videos, movies etc.) in that folder.
again run "key.bat" press y . done
your files are hidden. to get back your files whenever you want.

3. to unlock all you files:

put this script where you have locked and hide your files.
example: drive d > photos.
then run "key.bat" enter your password.

also you can create a multiple copies of this script for different drives, folders.
in case you deleted 'key.bat' script. then,
Download it from "https://github.com/ancod3r/HideFiles" and edit your password save it.
place downloaded script in that folder where you've hidden your files and run script.
Enter your password you will get back all your files.

4. troubleshoot:

    a. you want to unlock your files but this program creates a folder:

    solution:
    it means you haven't locked your files in that folder or drive.

    b. forgot your password:

    solution:
    open this program in notepad there you will find your password.
